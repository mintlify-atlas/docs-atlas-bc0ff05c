## System Requirements

Before installing Auth Dashboard, ensure your system meets these requirements:

- **Node.js**: Version 18.0.0 or higher
- **Package Manager**: npm, yarn, or pnpm
- **Operating System**: Windows, macOS, or Linux
- **Memory**: At least 4GB RAM for development

<Note>
  We recommend using the latest LTS version of Node.js for optimal performance and security.
</Note>

## Installation Methods

### Using npm

Install all dependencies using npm:

<CodeGroup>
  ```bash Install
  npm install
  ```

  ```bash Install with CI
  npm ci
  ```

  ```bash Clean Install
  rm -rf node_modules package-lock.json
  npm install
  ```
</CodeGroup>

### Using Yarn

Install all dependencies using Yarn:

<CodeGroup>
  ```bash Install
  yarn install
  ```

  ```bash Install Frozen
  yarn install --frozen-lockfile
  ```

  ```bash Clean Install
  rm -rf node_modules yarn.lock
  yarn install
  ```
</CodeGroup>

### Using pnpm

Install all dependencies using pnpm:

<CodeGroup>
  ```bash Install
  pnpm install
  ```

  ```bash Install Frozen
  pnpm install --frozen-lockfile
  ```

  ```bash Clean Install
  rm -rf node_modules pnpm-lock.yaml
  pnpm install
  ```
</CodeGroup>

<Tip>
  pnpm is the fastest package manager and uses less disk space due to its content-addressable storage.
</Tip>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Installation fails with permission errors">
    If you encounter EACCES errors, avoid using `sudo`. Instead:
    ```bash
    npm config set prefix ~/.npm-global
    export PATH=~/.npm-global/bin:$PATH
    ```
  </Accordion>

  <Accordion title="Peer dependency warnings">
    Peer dependency warnings are usually safe to ignore. If you want to resolve them:
    ```bash
    npm install --legacy-peer-deps
    ```
  </Accordion>

  <Accordion title="Module not found errors">
    Clear cache and reinstall:
    ```bash
    rm -rf node_modules package-lock.json
    npm install
    ```
  </Accordion>

  <Accordion title="TypeScript errors during build">
    Ensure TypeScript version matches:
    ```bash
    npm list typescript
    ```
    Should show version ~5.9.3
  </Accordion>

  <Accordion title="Vite build fails">
    Check Node.js version:
    ```bash
    node --version
    ```
    Vite 7 requires Node.js 18 or higher.
  </Accordion>
</AccordionGroup>
